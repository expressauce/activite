git init Active le dossier courant comme repository Git
git status Permet de connaître l'état des fichiers
git add [fichier] Ajoute un fichier, passé en paramètre, à l'index Git
git commit Enregistre les modifications effectuées repository.
-m "message" : permet de renseigner un message décrivant les modifications effectuées
git log Affiche la liste des commits réalisé
git clone [url] Clone un dépôt existant
git push origin master Envoi les modifs dans la branche master du remote origin
